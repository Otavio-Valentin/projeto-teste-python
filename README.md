# Auditoria de Ativos e Visualização com Python

Este projeto faz parte do meu laboratório de estudos em **Ciência de Dados**. O objetivo principal é demonstrar a manipulação de dados estruturados em formato JSON, o tratamento de variáveis e a geração de visualizações estatísticas para auditoria de ativos.

## 📊 O Projeto
O notebook `auditoria_json.ipynb` realiza o processamento de um conjunto de dados simulando ativos digitais (Logins, Cartões e Notas). 

### Funcionalidades:
- **Extração**: Leitura e carregamento de estruturas de dados.
- **Tratamento**: Limpeza e preparação dos dados com Pandas.
- **Visualização**: Geração de gráfico de pizza com Matplotlib para análise de proporção de categorias.

## 🛠️ Tecnologias Utilizadas
- **Python 3.14**
- **Pandas**: Para manipulação e análise de dados.
- **Matplotlib**: Para criação de visualizações gráficas.
- **Git & GitHub**: Para versionamento e integridade do código.

## 🧠 Desafios Superados
Durante o desenvolvimento, foquei na **reprodutibilidade** do código, garantindo que o fluxo lógico de execução (do topo ao rodapé) esteja livre de erros de dependência. Também foram aplicadas boas práticas de higiene de dados, mantendo os *outputs* do Jupyter Notebook limpos e organizados.
